About the Project

Project summary

Disease Prediction Using Machine Learning

MedScope is a Flask application that predicts a possible disease from selected symptoms using a Decision Tree Classifier trained on a small CSV dataset.

Python and Flask

Routes, sessions, account access, diagnosis records, and admin monitoring.

Decision Tree Classifier

Scikit-learn model trained from the symptom and disease CSV dataset.

Pandas CSV Workflow

The dataset is loaded, prepared, and converted into model features.

Clinical Portal Interface

The interface follows the Pathogenix clinical dashboard design direction from Stitch.

The project is for academic use and should not be used as medical advice.
